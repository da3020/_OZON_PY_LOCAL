# OZON_PY

Инструмент для получения заказов OZON в статусе `awaiting_packaging`
и формирования производственного списка.

## Текущая версия
v6.3.6 ALT

## Возможности
- мультиаккаунты OZON
- кэш категорий товаров
- вывод:
  - терминал
  - HTML
  - Excel
- без изменения статусов заказов

## Запуск
```bash
python src/awaiting_packaging.py
